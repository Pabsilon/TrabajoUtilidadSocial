¿Qué nos ofrece Github siendo estudiantes?


Student developer pack


Si eres estudiante, no te puedes perder lo que ofrece GitHub Student Developer Pack, acceso gratis durante un tiempo o para todo el periodo de estudiante, a una buena cantidad de herramientas muy útiles para desarrolladores.

Es un paquete muy interesante, ya que nos ofrece por ejemplo, mientras seas estudiante cuentas de GitHub de pago, unos 7 dólares al mes, donde podremos tener hasta 5 repositorios privados, una suscripción a Unreal Engine o un año gratis de Bitnami,... pero para disfrutar de ello tendrás que registrarte con el correo electrónico del centro donde estudies, pasamos a detallar las ventajas:

  1. ATOM
    Uno de los mejores editores de texto

  2. AWS Cloud
    Plataforma de servicios en la nube

  3. Bitnami
    Para facilitarnos el desarrollo web

  4. Carto
    Plataforma de análsis de datos, virtualización y creación de aplicaciones

  5. CrowdFlower
    Plataforma para análisis de datos

  6. Datadog
    Servicio de monitoreo para aplicaciones

  7. Digital Ocean
    Para poder montar un servidor linux muy facilmente

  8. Flatiron School
    Nos ayuda a aprender desarrollo web

  9. GitHub
    Cuenta micro

  10. GitKraken
    Un cliente de Git para Windows/Mac/Linux

  11. Hackhands
    Nos ayudan a programar, disponibles 24/7

  12. Microsft Imagine
    Nos ofrece Azure y otras herramientas de desarrollo

  13. Namecheap
    Nos dan un dominio .me durante 1 año

  14. SendGrid
    Nos ofrece una infraestructura solamente para correos electrónicos

  15. Sentry
    Raestrea errores en nuestros proyectos

  16. Stripe
    Pagos web y móvil

  17. Taplytics
    Plataforma para aplicacioens móviles

  18. Thinkful
    Nos ayudan con la programación web

  19. Transifex
    Ayuda a traducir nuestros proyectos

  20. Travis CI
    Plataforma muy conocida para proyectos tanto privados como open source

  21. Udacity
    Nos ofrece cursos y clases de todo tipo

  22. Unreal Engine
    Herramienta para desarrollo de juegos en PC, consola, web...
